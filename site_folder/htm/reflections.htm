Reflections



















While I greatly enjoy consuming stories, I don't derive anywhere near as much pleasure from creating them.
I found, as was appropriate, that building a website for the Web Development class was a learning experience.
Learning how to edit and make changes to existing pages throughout the quarter helped me with the nuts and bolts part of building
this website, but if you don't have content for the website, it is difficult to develop the plan for how to come up with the content.
I'm finding that, in general, I still prefer organizing, arranging, and editing raw content developed by someone else over creating my own content.
